## Get Events

<div className="grid grid-cols-1 xl:grid-cols-2 gap-8">
<div>

```
GET /api/sites/:site/events
```

Returns a paginated list of events (pageviews, custom events, and outbound clicks).

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    page_size: {
      description: 'Number of results per page',
      type: 'number',
      default: 50,
    },
    since_timestamp: {
      description: 'Poll for new events after this timestamp (ISO 8601). Used for realtime polling. Returns up to 500 events.',
      type: 'string',
    },
    before_timestamp: {
      description: 'Fetch events before this timestamp (ISO 8601). Used for cursor-based pagination (scrolling back).',
      type: 'string',
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of event objects',
      type: 'Event[]',
    },
    cursor: {
      description: 'Cursor metadata for pagination (only present when not using since_timestamp)',
      type: '{ hasMore: boolean, oldestTimestamp: string | null }',
    },
  }}
/>
