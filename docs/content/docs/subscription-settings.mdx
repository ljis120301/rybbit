import { Callout } from 'fumadocs-ui/components/callout';

Access subscription settings by hovering over the sidebar and clicking "Subscription".

Subscriptions are scoped to a single organization. If you have multiple organizations, each one requires its own subscription. Switching organizations will show that organization's subscription status.
